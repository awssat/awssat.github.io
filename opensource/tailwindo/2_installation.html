    # Installation

    To get started, use Composer to install the tool in your system:

    composer global require awssat/tailwindo

    TIP

    To install this tool make sure that PHP & Composer (a PHP package manager) are installed on your system .

    or you can use NodeJs version to install the tool in your system (a seperate project, not affiliated with Awssat):

    npm i -g tailwindo

    TIP

    To install the nodejs version of the tool make sure that NodeJs > 10 & NPM (a Node Package Manager) installed on your system .
